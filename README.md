# Medicare E-Commerce Website

## 📖 About
This is a simple Medicare E-Commerce website developed using PHP and MySQL. It allows users to browse medicines, add products to the cart, and place orders online. An admin can manage products and customer orders.

## ✨ Features
- User Registration & Login
- Browse Medicines
- Search Products
- Add to Cart
- Place Orders
- Admin Dashboard
- Manage Products & Orders

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- XAMPP

## 🚀 How to Run
1. Install XAMPP.
2. Copy the project folder to `xampp/htdocs`.
3. Start Apache and MySQL from XAMPP.
4. Import the database into phpMyAdmin.
5. Open your browser and visit:
   ```
   http://localhost/AmanMedicare
   ```
